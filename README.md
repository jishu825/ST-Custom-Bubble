# SillyTavern Chat Stylist

为 SillyTavern 聊天界面提供全面的样式定制功能，包括消息气泡、文本样式和特效等个性化设置选项。

## 功能特点

- 个性化消息气泡样式
  - 背景颜色和透明度
  - 渐变效果
  - 边框样式
- 文本样式定制
  - 主要文本颜色
  - 斜体文本样式
  - 引用文本效果
  - 文本荧光效果
- 每个角色独立样式
- 实时预览
- 样式导入/导出

## 安装方法

1. 打开 SillyTavern 的扩展页面
2. 点击"安装扩展"
3. 输入此仓库的 URL
4. 点击保存
